kueue-operator: add e2e test for disconnected env

This commit adds step registry configuration for kueue-operator disconnected environment testing:

- Compute step: resolves bundle, operator, and must-gather images using skopeo
- Catalog step: mirrors images to disconnected registry and sets up image mirroring
- Operator step: installs operator using catalog subscription approach with mirrored bundle image
- Image patch step: updates CSV to use mirrored operator image
- Added documentation to all ref.yaml files using |- format
